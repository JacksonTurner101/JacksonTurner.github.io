CPU and GPU

This project included using the CPU and GPU to create a particle simulation. Essentially, there is a spray can spraying paint particles on a peice of paper. After the simulation has finished there is an output for the paper that the spray can was aiming at. One simulation is on the CPU using the programming language Rust. There is another simulation on the GPU using CUDA.
